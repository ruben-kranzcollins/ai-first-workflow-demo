Demonstrate the team workflow by making a small change to `src/demo.txt` in a separate branch and opening a pull request.

## Why

This issue exists to walk through the full flow once: issue → branch → pull request → review → merge.

## Acceptance criteria

- [ ] Change is made in a branch, not directly on `main`
- [ ] A pull request is opened with the PR template filled in
- [ ] AI use is disclosed in the pull request
- [ ] The safety checklist is completed (no secrets, no PII, no client data)
- [ ] At least one human reviewer approves
- [ ] The PR is merged into `main` and this issue closes automatically
